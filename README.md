# AgeOfClickSons

 **Jeu réalisé lors de la Code Game Jam 2026**  
 Arrivée **2ᵉ a l'IUT d'orsay (au niveau local)**

---

## Description

AgeOfClickSons est un jeu de type *clicker / stratégie légère* développé avec **Godot Engine**.  
Le but du jeu est de gérer vos ressources, d’envoyer vos alliés et de détruire le chapiteau adverse pour gagner la partie.

---

## Gameplay

- **Éclater des ballons** pour récupérer des ressources.   
- **Spawner des alliés** pour attaquer le chapiteau adverse.   
- **Easter eggs** cachés à découvrir pour des bonus amusants.  
- **Perdre la partie** si votre chapiteau est détruit. 
- **Gagner la partie** en détruisant le chapiteau de l’adversaire, contrôlé par une IA basique. 

### IA et difficulté

- L’adversaire spawn des unités selon la vague.  
- Tous les **10 vagues**, un **boss** apparaît.  
- Le comportement de l’IA est **fixe** : pas d’adaptation à vos actions.
